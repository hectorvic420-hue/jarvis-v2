Diseño del Landing Builder de Jarvis

Revisa la arquitectura completa — dime si algo no te convence antes de implementar

🧠 Flujo completo

Usuario → Jarvis → Landing publicada en internet
1. Usuario pide landing
"Crea una landing para mi curso de Marketing Digital, $197, Hotmart"
O: { title, description, style, checkout_url, ... }

2. landing_builder tool (Jarvis)
• Parsea el prompt libre con Claude
• Determina: título, estilo, secciones, checkout_url
• Llama a Claude con LANDING_EXPERT_PROMPT

3. Claude genera HTML experto
• HTML/CSS/JS en un solo archivo autocontenido
• Todas las secciones de la opción C
• Estilo elegido (futurista / premium / energético / etc.)

4. Publish al servidor GCP
• Guarda en /opt/jarvis/landings/[slug].html
• Express sirve la carpeta en /l/[slug]
• Retorna URL pública: https://tudominio.com/l/curso-marketing

⚙️ Parámetros de la tool

Qué le puedes pedir a Jarvis
action: create_landing | list_landings | delete_landing
prompt: "Crea una landing para mi curso de IA para emprendedores"
style: futuristic | premium | energetic | corporate | natural | bold | auto
checkout_url: https://pay.hotmart.com/XXX (cualquier URL)
pixel_id: Meta Pixel ID (opcional)
ga_id: Google Analytics ID (opcional)
countdown_hours: 24 (optional urgency timer)
